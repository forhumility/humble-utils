# Humble Utils

> A humble utils

## Setup

### install humble utils

``` bash
npm i humble-utils --save
```

### import date format

``` javascript
import { format } from "humble-utils";
```

### using date format

``` javascript
let fmt;
fmt = format(new Date()); // "1996-07-22 13:14:00"
fmt = format(new Date(),"MM-dd"); // "07-22"
```
