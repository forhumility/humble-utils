# Humble Utils

> A humble utils

## Setup

### install humble utils

``` bash
npm i humble-utils --save
```

### import date dateFormat

``` javascript
import { dateFormat } from "humble-utils";
```

### using date dateFormat

``` javascript
let fmt;
fmt = dateFormat(new Date()); // "1996-07-22 13:14:00"
fmt = dateFormat(new Date(),"MM-dd"); // "07-22"
```
